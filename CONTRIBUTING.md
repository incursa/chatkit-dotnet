# Contributing

Thanks for helping improve this library.

- Keep changes focused and reviewable.
- Follow the existing .NET and Razor conventions in the repository.
- Run the relevant tests before opening a pull request.
- Unless a file already carries a different upstream notice, treat contributions as licensed under the repository MIT license.
- No contributor agreement is required for routine pull requests.

For source files, a short header is acceptable when you want explicit authorship and license markers:

```csharp
// Copyright (c) 2026 Incursa
// SPDX-License-Identifier: MIT
```

If you add a header to a `.cshtml` or `.razor` file, use a Razor comment block instead:

```razor
@* Copyright (c) 2026 Incursa
   SPDX-License-Identifier: MIT *@
```
